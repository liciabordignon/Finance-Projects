# Financial-Analysis
This repository contains a collection of projects developed as part of the *Finance with Big Data* course. It showcases the application of data science, statistical methods, and machine learning techniques to real-world financial problems.

## Repository Structure

- `Lab1.ipynb` – Portfolio construction and basic financial analysis  
- `Lab2.ipynb` – Risk analysis and asset pricing (e.g., beta estimation)  
- `Lab3.ipynb` – Sentiment analysis of financial text data (Twitter)  
- `Lab4.ipynb` – Machine learning models for stock return prediction  
- `Hackathon.ipynb` – ESG and event study analysis using NGO campaign data  
- `data/` – Datasets used across the different projects  

## Topics Covered

- Portfolio optimization (Tangency portfolio)
- Risk modeling and CAPM
- Sentiment analysis (NLP on financial text)
- Machine learning for return prediction
- Time series analysis of financial data
- ESG and responsible investing
- Event study methodology (CAR analysis)
